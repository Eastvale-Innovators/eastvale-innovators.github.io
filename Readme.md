## Important contribution policy
**Only approved contributors may push directly. All others should open a GitHub Issue describing proposed changes.**

If you're not an approved contributor, open an Issue with:
- A clear summary of the proposed change
- Files/lines affected (if known)
- Any screenshots, mockups, or previews

Maintainers will review and either:
- Request changes, or
- Ask you to submit a branch/PR, or
- Grant push access for the change

## For approved contributors
- Pull the latest main before working: `git pull origin main`
- Create a feature branch for non-trivial work: `git checkout -b feature/your-change`
- Push following repo policy and CI requirements
